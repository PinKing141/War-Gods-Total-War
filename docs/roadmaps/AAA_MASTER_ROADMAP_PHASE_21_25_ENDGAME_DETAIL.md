# Phases 21-25 endgame roadmap detail

## Phase 21. World Scale Gate 2: 35-50 political entities

Subphases:

```text
21A Regional minor states
21B Regional conflict web
21C 100-year balance test
```

Closing gate:

```text
The world reaches 35-50 entities and every region has political, resource, identity, institution and non-state tension.
```

## Phase 22. Scenario tools, debug controls and modding pipeline

Subphases:

```text
22A Scenario tools
22B Developer debug controls
22C Balance dashboard
22D Timeline replay
22E Modding/data pipeline
```

Closing gate:

```text
The developer can control, test, replay, balance and expand the world safely.
```

## Phase 23. World Scale Gate 3: 100+ political entities

Subphases:

```text
23A 100+ entity data expansion
23B Content density pass
23C Long-run world stability
```

Closing gate:

```text
The world has CK-scale political density, uses tiered simulation and survives 100/250/500-year tests.
```

## Phase 24. Presentation, art polish and player-facing feel

Subphases:

```text
24A Map art pass
24B Rivers and trees final pass
24C Icon and marker system
24D UI polish
24E Sound, ambience and onboarding
```

Closing gate:

```text
The game is readable, polished and understandable to a new observer-mode player.
```

## Phase 25. Release, community and long-term support

Subphases:

```text
25A Stability and performance
25B Balance and tuning
25C Scenario packaging
25D Documentation
25E Public build
```

Closing gate:

```text
The public build is stable, documented, balanced enough for testing and hides debug tools by default.
```
