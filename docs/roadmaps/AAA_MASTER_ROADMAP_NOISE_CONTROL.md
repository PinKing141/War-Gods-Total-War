# Roadmap noise control

This roadmap is allowed to be detailed, but implementation tasks should stay small.

## Good Codex task size

```text
One subphase gate
One file family
One validation/check area
One UI copy pass area
One simulation system at a time
```

## Bad Codex task size

```text
Add characters, dynasties, diplomacy, economy and naval warfare all at once.
Rewrite all content while adding 100 countries.
Polish the map while implementing validation.
Add UI, simulation and data for six systems in one change.
```

## Rule

The roadmap can be huge. Each actual implementation task should be narrow.
