## Added system interlink rules

Every new system should connect into existing systems.

Examples:

```text
characters -> dynasties -> succession -> claims -> war
institutions -> law -> taxes -> unrest -> revolts
culture -> identity tension -> diplomacy -> rebellion
faith -> legitimacy -> religious war -> chronicles
trade -> resources -> economy -> diplomacy -> war goals
ports -> fleets -> sea routes -> blockades -> coastal sieges
magic -> institutions -> faith tension -> disasters -> dynasties
```

## Added naval subphase structure

Naval is split into gated steps:

```text
20A Port system
20B Fleet objects
20C Fleet missions
20D Sea routes and blockades
20E Army transport and amphibious movement
```

This prevents jumping straight into naval warfare before ports, trade and fleets exist.

## Added release readiness requirements

The game is not release-ready until:

```text
validation passes
normal UI has no debug wording
100/250/500-year simulations pass
save/load survives major events
characters and dynasties affect history
diplomacy and economy affect war choices
naval systems matter on coasts
chronicle produces readable history
scenario tools work
public build hides debug by default
documentation exists
```
